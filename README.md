# Microservice Auth

This microservice handles **user authentication** for the clothing startup project.  

Features:
- User registration (email, password, name)  
- User login with **JWT** generation  
- Credential validation  
- Security using **JWT_SECRET** and hashed passwords with bcrypt  

## Technologies
- Node.js
- Express
- Prisma ORM
- PostgreSQL
- bcrypt
- JSON Web
